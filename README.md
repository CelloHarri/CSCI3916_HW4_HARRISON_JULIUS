# How To Post Review

You must first use Get Movie which stores the movie Id in an environment Variable. Then You may post a review using POST Review. You can change the review in the JSON Body

When You Post a Review, it stores the review ID as an environmental Variable for easier testing

When You GET a singular review with a an ID it stores reviewID as an environmental variable (doesn't matter now, but when interacting with the website, this will allow you to delete the review by GETing the review)
